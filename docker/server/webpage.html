California Digital Vaccine Record Table

Generating a California Digital Vaccine Record Table

The input is a ZIP file containing images of California digital vaccine records or files with the extension "shc" containing Smart Health Card URIs. The filename must end in the suffix ".zip" if a browser cannot determine that the media type is application/zip and classifies it as application/octet-stream instead.

After downloading, save the CSV file that was generated or open it with a suitable application. A spreadsheet should be able to open it. If there are options for delimiters separating fields, choose commas.
